自分用ドットファイルバックアップ
====

Overview  
バックアップは大切だよ!!  
## Description
vim、zshとgitの設定ファイルをバックアップしよう！  
## Usage
```
cd ~  
git clone git@github.com:nebchadnezzar/dotfiles.git  
```

## Instal
```
chmod +x dotfilesInstaller.sh  
./dotfilesInstaller.sh  
```
